Core Principles

Evolution = Structure: Lower stages are formless and ethereal. Higher stages gain geometric definition and architectural precision.

Thematic Cohesion: All forms must feel like "digital beings" — glowing, semi-transparent, data-adjacent. No solid/opaque forms.

Creative Freedom within Bounds: Each stage defines constraints (opacity range, element count, blur radius), not specific implementations.

  • Nascent: Orb, wisp, rune, glyph — any formless light
  • Signal: Hexagon, circle, triangle, diamond — simple geometry
  • Resonant: Monitor, hologram, crystal, lantern — defined character
  • Manifest: Clock, mandala, lattice, mechanism — complex architecture
  • Anchored: Shrine, totem, workstation, altar — fully realised presence

Preserved Identity Through Evolution

As an Ehko evolves, these elements persist to maintain continuity:

  • Colour Hue: Gradual stage-based shifts (Blue → Green → Gold → Pink → Bronze)
  • Seed-Derived Quirks: Unique parameter values (antenna angles, eye spacing, detail patterns)
  • Bilateral Symmetry: Left/right balance maintained
  • Central Anchor: Core element always near center point (40, 40)
